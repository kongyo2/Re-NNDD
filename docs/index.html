Re:NNDD

ニコニコ動画専用デスクトップアーカイブクライアント
Tauri 2 · Rust · Svelte 5 · TypeScript

🐧 Linux 🍎 macOS 🪟 Windows MIT License
⬇ Download GitHub Discord

🔍 スナップショット検索

ニコニコ動画のスナップショット検索 API 経由で高速に動画を検索。

▶️ HLS 再生 & コメント

内蔵プレイヤーで HLS 動画を再生し、コメントをリアルタイム表示。

⬇️ ダウンロード & ライブラリ

yt-dlp + ffmpeg で動画を保存し、SQLite ベースのライブラリで一元管理。

📝 コメント焼き込み

コメントを ASS 字幕として映像に焼き込んだ MP4 をエクスポート。

📋 プレイリスト

マイリストとスマートプレイリストで動画を整理。

🚫 NG 機能

NG ルールで不要なコンテンツをフィルタリング。

🕹️ コマンドパレット

キーボード操作で素早く機能にアクセス。

🔌 プラグインシステム

JavaScript プラグインで機能を拡張可能。
