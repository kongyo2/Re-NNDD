ニコニコ動画専用デスクトップアーカイブクライアント

Re:NNDD
ニコニコ動画デスクトップアーカイブクライアント
開発者 abeshinzo78
初版 2025年
対応 OS Linux / macOS / Windows
種別 動画ダウンロード・再生ソフト
ライセンス MIT
使用言語 Rust, TypeScript, Svelte

Re:NNDD（リ・エヌエヌディーディー）は、ニコニコ動画専用のデスクトップアーカイブクライアントである。 Tauri 2 + Rust + Svelte 5 + TypeScript で構築されており、Windows・macOS・Linuxに対応する。 NNDDオリジナルの著者 MineAP 氏の MIT ライセンス NNDD を起点に、現代的なスタックで再実装するプロジェクトである。

機能

  • スナップショット検索ニコニコ動画のスナップショット検索 API 経由で高速に動画を検索
  • ログインメール/パスワード、user_session Cookie 直入力に対応
  • HLS 再生 & コメント内蔵プレイヤーで HLS 動画を再生し、コメントをリアルタイム表示
  • ダウンロード & ライブラリyt-dlp + ffmpeg で動画を保存、SQLite ベースのライブラリで一元管理
  • コメント焼き込みコメントを ASS 字幕として映像に焼き込んだ MP4 をエクスポート
  • ランキング総合・VOCALOID・ゲーム・アニメ・音楽など各ジャンル別ランキングを表示
  • ユーザー/チャンネルユーザー・チャンネルの投稿動画一覧を取得
  • プレイリストマイリストとスマートプレイリストで動画を整理
  • NG 機能NG ルールで不要なコンテンツをフィルタリング
  • コマンドパレットキーボード操作で素早く機能にアクセス
  • プラグインシステムJavaScript プラグインで機能を拡張可能
  • ローカル再生内蔵 HTTP Range 配信でローカル保存動画を再生

インストール

Releases から各プラットフォーム向けのバイナリをダウンロードできる。

開発版をソースからビルドする場合:

git clone https://github.com/abeshinzo78/Re-NNDD.git
cd Re-NNDD
npm install
bash scripts/fetch-binaries.sh
npm run tauri:dev
